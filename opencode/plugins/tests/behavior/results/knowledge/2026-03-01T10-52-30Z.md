You have access to the following tools:

## Core File Operations
- **read**: Read a file or directory from the local filesystem
- **write**: Create or overwrite a file
- **edit**: Perform exact string replacements in files
- **morph-mcp_edit_file**: Fast editing for large files with scattered changes using placeholder comments

## Search & Exploration
- **grep**: Fast content search using regular expressions
- **glob**: Fast file pattern matching
- **warpgrep** (morph-mcp_warpgrep_codebase_search): Exploratory codebase search using natural language queries
- **serena_search_for_pattern**: Flexible pattern search with context control

## Symbol & Code Analysis (Serena-specific)
- **serena_get_symbols_overview**: Get high-level understanding of code symbols in a file
- **serena_find_symbol**: Find symbols by name path pattern
- **serena_find_referencing_symbols**: Find where a symbol is referenced
- **serena_replace_symbol_body**: Replace a symbol's implementation
- **serena_insert_after_symbol**: Insert code after a symbol definition
- **serena_insert_before_symbol**: Insert code before a symbol definition
- **serena_rename_symbol**: Rename a symbol throughout the codebase

## Shell Operations
- **bash**: Execute shell commands with timeout control
- **serena_execute_shell_command**: Execute shell commands with configurable options

## Project & Memory Management
- **serena_activate_project**: Activate a project
- **serena_read_memory**: Read stored project memories
- **serena_write_memory**: Write information to project memory
- **serena_list_memories**: List available memories
- **serena_delete_memory**: Delete a memory
- **serena_rename_memory**: Rename or move a memory

## Web Research
- **kindly_web_search**: Search the web and return top results with markdown content
- **kindly_get_content**: Fetch a specific URL and return best-effort markdown
- **webfetch**: Fetch content from a URL (alternative)

## Task Delegation
- **task**: Launch specialized subagents for complex, multi-step tasks (general, explore, code-reviewer, test-engineer, etc.)

## Planning & Coordination
- **write_plan**: Commit a plan to disk
- **todowrite**: Create and manage structured task lists
- **question**: Ask the user questions during execution

## Utilities
- **introspection**: Get your own session ID and agent info
- **list_sessions**: List OpenCode sessions
- **read_transcript**: Read a session transcript
- **sleep**: Wait for a duration
- **sleep_until**: Wait until a specific time
- **async_command**: Run background tasks
- **tokenscope**: Analyze token usage

## Clipboard Operations (Cut/Copy/Paste MCP)
- **cut-copy-paste-mcp_copy_lines**: Copy lines to session clipboard
- **cut-copy-paste-mcp_cut_lines**: Cut lines to session clipboard
- **cut-copy-paste-mcp_paste_lines**: Paste from clipboard
- **cut-copy-paste-mcp_show_clipboard**: Display clipboard contents
- **cut-copy-paste-mcp_undo_last_paste**: Undo last paste
- **cut-copy-paste-mcp_get_operation_history**: Get clipboard operation history

## Library Documentation
- **context7_resolve-library-id**: Resolve a library name to Context7 ID
- **context7_query-docs**: Query up-to-date documentation using Context7

## Skills System
- **skill**: Load specialized skills that provide domain-specific instructions and workflows (python, clean-code, test-guidelines, systematic-debugging, etc.)

Each tool is designed for specific tasks—from file manipulation and code analysis to web research and subagent orchestration. Use the appropriate tool for the job, and load relevant skills when applicable.
