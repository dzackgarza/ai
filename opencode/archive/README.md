# Archive

This directory is for historical material that should not compete with the
canonical workflows.

## Contents

- `provider-scripts/`: superseded provider validation experiments and one-off probes
- `runtime-artifacts/`: stray logs, binaries, and scratch outputs that do not belong at the repo root

## Rule

If a file lives here, it is not the supported pathway. Either replace it with a
recipe under `just`, a maintained script near its owning subtree, or leave it
archived.
