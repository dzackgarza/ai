# suggested commands
- Repo orientation: `find . -maxdepth 2 \( -name README.md -o -name AGENTS.md \) | sort`
- Rebuild permissions into config fragments: `python3 manage_permissions.py --apply`
- Rebuild merged config artifact: `python3 build_config.py`
- Plugin checks: `cd plugins && just check`
- Plugin classifier tests: `cd plugins && just classifier` or `just classifier-model <provider/model>`
- Plugin behavioral tests: `cd plugins && just behavior A` or `just baseline A`
- Harness help: `cd harness && bun run opx --help`
- Session harness via plugin recipes: `cd plugins && just session list`
- File inventory: `rg --files configs plugins harness scripts | sort`
- Git diff review: `git diff` and `git diff --cached`
