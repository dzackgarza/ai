# Reading PDFs

Start here for any PDF request. Select one leaf procedure:

- [[reading-pdfs/ocr/SKILL|OCR reading]] — turn a PDF into cached Markdown with Mistral OCR before reading it.
- [[reading-pdfs/extraction/SKILL|structured extraction]] — extract text, tables, figures, captions, or a higher-fidelity artifact for a paper or report.

Do not load both leaves unless the request requires both outputs. Read the selected leaf before conversion or extraction.
