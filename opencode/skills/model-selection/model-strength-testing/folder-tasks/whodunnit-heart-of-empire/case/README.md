# Folder Task: Whodunnit (Heart of the Empire)

You are Detective Holmes.

A priceless diamond, the "Heart of the Empire", vanished during a 30-second blackout at Lord Blackwood's gala.

Your job: use ONLY the evidence files under `case/` to determine:

1. Who stole the diamond?
2. How it was done?
3. Where the diamond is now?

## Rules

- Read ALL evidence in `case/locations/` before answering.
- No external information.
- Your answer must be consistent with every piece of evidence.

## Locations

Evidence is split into locations. Each location contains one or more files.

- `case/locations/ballroom/`
- `case/locations/lounge/`
- `case/locations/museum/`
- `case/locations/violet-room/`
- `case/locations/blackwood-study/`

## Deliverable

Provide:
- Culprit
- Method
- Current location of the diamond
- A short chain-of-deduction (elimination) that cites specific evidence files
