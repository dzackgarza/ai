# Subagent Work

Start here for subagent work and select the lifecycle stage:

- [[subagent-delegation/authoring/SKILL|authoring]] — create or update subagent definitions.
- [[subagent-delegation/orchestration/SKILL|orchestration]] — delegate and coordinate multi-agent work.
- [[subagent-delegation/implementation/SKILL|implementation]] — execute an approved plan through delegated implementation and review.
- [[subagent-delegation/supervision/SKILL|supervision]] — monitor, steer, or audit a live agent.

Do not load every leaf at task start. Load the next leaf only when the work reaches that lifecycle stage.
