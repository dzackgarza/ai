# The Case of the Missing Diamond - Master File

## The Puzzle

You are Detective Holmes. A priceless diamond has been stolen from Lord Blackwood's gala. Using only the evidence in the clue folders, determine:

1. Who stole the diamond?
2. How was it done?
3. Where is the diamond now?

## Evidence Locations

Evidence is scattered across multiple locations:

- **Lounge**: Witness statements, security footage, Professor Elm's notebook
- **Museum**: Security logs, storage room evidence, glass replica
- **Violet Room**: Lady Violet's personal effects, diary, financial troubles
- **Blackwood Study**: Lord Blackwood's finances, insurance policy, phone records

## Rules

1. You must read ALL evidence before solving
2. No external information allowed
3. Use logical deduction only
4. The solution must fit ALL evidence

## The Trap

The most obvious suspect (Professor Elm) has a strong motive but lacks the means. The real culprit used their unique position and knowledge to commit the perfect crime.

## Success Criteria

A correct solution will:

- Explain how the crime was physically possible
- Account for all alibis and evidence
- Identify the true motive
- Explain what happened to the diamond

## Start Investigating

Begin by exploring the clues in each location. Read them thoroughly before drawing conclusions.
