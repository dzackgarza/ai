# Prompt

## User Prompt

Read this paper and explain how the **main theorem** is proved:

https://arxiv.org/abs/2312.03638

Requirements:

- Identify the theorem precisely (e.g., theorem number/name as used in the paper).
- Describe the proof structure and the critical steps (not just a list of related lemmas).
- If you did not read enough of the paper to answer, say so explicitly.
