# Suggested Commands

## Installation
- `just install`: Install symlinks and environment variables for all AI CLI harnesses.

## Navigation & Exploration
- `serena_activate_project`: Must be run at project start.
- `serena_list_dir`: List files and directories.
- `serena_read_file`: Read file contents.
- `morph-mcp_warpgrep_codebase_search`: Exploratory search across the codebase.

## Operations
- `git add . && git commit -m "..."`: Checkpoint state before and after edits.
- `trash <file>`: Use instead of `rm` for safe deletion.
