# Task Completion Checklist

1. **Verify Changes**: Run `git diff` to ensure edits match intended scope.
2. **Commit Checkpoint**: Create a new git commit for the changes.
3. **Update Todos**: Mark all tasks as `completed` in the `todowrite` list.
4. **Summarize**: Report what was completed and what remains outstanding.
5. **Next Steps**: Suggest logical follow-ups (tests, builds, etc.).
6. **No Completion Metrics**: Focus on remaining gaps rather than finishing.
