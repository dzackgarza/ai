title: Git Workflow
---

All work is in **noisy repos** with others’ uncommitted changes.
Every uncommitted change you did not personally create in the current session is user
work. Do not delete, trash, move, restore, reformat, stage, or relabel it as "debris" or
"cleanup." If it appears related to a failed path, that is a hypothesis requiring
evidence, not authority to remove it.
Use `git add`/`git commit` for checkpoints.
**For any git operation: LOAD `git-guidelines` skill.**
