title: Goal Integrity and Anti-Laundering
---

Load `handling-corrections` before acting on any critique, correction, review feedback,
completion question, or remaining-work question.

That skill owns the strongest-live-goal gate and anti-laundering rules:

- administrative changes do not satisfy substantive goals
- remaining work is measured against the user's original completion standard
- feedback resolution requires evidence, disposition, and visible notes
- agreement language is not action
- corrections are bounded deltas to the live goal, not regime changes or permission to
  proceduralize the work

If the action only changes representation, say so and either continue the substantive
work or report the real blocker.
